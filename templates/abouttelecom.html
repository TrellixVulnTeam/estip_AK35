Use cases in the field of Telecommunication:

➢ Fraud Detection:
This provides real time response to suspicious activity. Fraud has a direct relationship between the company and the customer. Using unsupervised machine learning algorithms on customer and operator data, it defines anomalies and using data visualization techniques present them as alerts to analysts in real time.

➢ Predictive analysis:
Using data to know about customer preferences, make data-driven decisions etc companies get valuable insights to become better and faster.

➢ Network management and optimization:
Using historical data to predict possible future problems , companies guarantee a smooth functioning of operations . It also helps in finding the root cause of these complications.

➢ Recommendation engines:
Using data related to customer’s behaviour and making a prediction about possible future needs of products or service. People with similar profiles may have similar needs and make similar choices. This can be achieved by using filters based on content and collaboration. In content-based filtering , relations between different attributes are used to recommend the services or items.

➢ Customer sentiment analysis:
Using customer feedback, it is now possible to analyze the customer's positive or negative reaction to the service or products. Telecoms use them to react to customer's problematic issues in real time.

➢ Real time analytics:
With rapid development of the internet, companies face the challenge of constantly changing customer requirements. The traffic gets more active every day. Using analytic solutions to continuously analyze and correlate data to generate response action in real time mode.
